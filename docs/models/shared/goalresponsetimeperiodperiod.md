# GoalResponseTimePeriodPeriod

The cadence and index of the time period. The value is one of: `FY`, `H1`, `H2`, `Q1`, `Q2`, `Q3`, or `Q4`.


## Values

| Name  | Value |
| ----- | ----- |
| `FY`  | FY    |
| `H1`  | H1    |
| `H2`  | H2    |
| `Q1`  | Q1    |
| `Q2`  | Q2    |
| `Q3`  | Q3    |
| `Q4`  | Q4    |

# AuditLogEventContextContextType

The type of context.
Can be one of `web`, `desktop`, `mobile`, `asana_support`, `asana`, `email`, or `api`.


## Values

| Name            | Value           |
| --------------- | --------------- |
| `WEB`           | web             |
| `DESKTOP`       | desktop         |
| `MOBILE`        | mobile          |
| `ASANA_SUPPORT` | asana_support   |
| `ASANA`         | asana           |
| `EMAIL`         | email           |
| `API`           | api             |

# JobResponseStatus

The current status of this job. The value is one of: `not_started`, `in_progress`, `succeeded`, or `failed`.


## Values

| Name          | Value         |
| ------------- | ------------- |
| `NOT_STARTED` | not_started   |
| `IN_PROGRESS` | in_progress   |
| `SUCCEEDED`   | succeeded     |
| `FAILED`      | failed        |

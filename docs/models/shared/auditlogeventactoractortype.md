# AuditLogEventActorActorType

The type of actor.
Can be one of `user`, `asana`, `asana_support`, `anonymous`, or `external_administrator`.


## Values

| Name                     | Value                    |
| ------------------------ | ------------------------ |
| `USER`                   | user                     |
| `ASANA`                  | asana                    |
| `ASANA_SUPPORT`          | asana_support            |
| `ANONYMOUS`              | anonymous                |
| `EXTERNAL_ADMINISTRATOR` | external_administrator   |
